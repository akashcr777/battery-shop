## üéØ How It Works

### Image Priority System

The app will try to load images in this order:

1. **Product-specific image** ‚Üí `/images/products/{product-filename}.jpg`
2. **Category fallback** ‚Üí `/images/battery-two-wheeler.jpg` (etc.)
3. **Beautiful placeholder** ‚Üí Auto-generated SVG with product name

### Example for "Exide Xplore XLTZ4A (4Ah)":

```
Try 1: /images/products/exide-xplore-xltz4a-4ah.jpg  ‚Üê Product-specific
       ‚Üì (if not found)
Try 2: /images/battery-two-wheeler.jpg              ‚Üê Category fallback
       ‚Üì (if not found)
Try 3: Beautiful SVG placeholder                     ‚Üê Auto-generated
```

## üí° Image Tips

### Product-Specific Images
- **Best for:** Featured products, new arrivals, premium items
- **Source:** Manufacturer websites, product catalogs
- **Benefit:** Customers see exactly what they're buying

### Category Images
- **Best for:** Quick setup, similar-looking products
- **Source:** Generic battery/UPS photos
- **Benefit:** Fast implementation, fewer images needed

### File Naming Rules

The app automatically converts product names to filenames:
- Converts to lowercase
- Replaces spaces with hyphens
- Removes special characters
- Example: `"Exide XPLORE XLTZ4A (4Ah)"` ‚Üí `exide-xplore-xltz4a-4ah.jpg`
